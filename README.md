- **Сборка:** Gradle (Kotlin DSL)

## Краткое описание приложения
Интернет-магазин. 
Приложение позволяет просматривать товары, добавлять их в корзину и оформлять заказы.  
Разрабатывается в рамках курса мобильной разработки.
